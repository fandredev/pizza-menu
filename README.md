## Clone this repo

```
git clone git@github.com:fandredev/pizza-menu.git
```

## Go to directory

```
 cd pizza-menu
```

## Install dependencies

```
 npm install -f
```

## Run server

```
 npm run start
```

## Run tests

```
 npm run test
```
